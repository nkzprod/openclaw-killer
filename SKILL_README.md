# OpenClaw Killer Skill

一个讽刺性的 OpenClaw Skill - 让 OpenClaw 卸载自己 🦞

## 安装

```bash
# 克隆仓库
git clone https://github.com/orange2ai/openclaw-killer.git
cd openclaw-killer

# 安装 Skill
./install-skill.sh
```

## 使用

在 OpenClaw 中说：
- "卸载 OpenClaw"
- "删除 OpenClaw"
- "移除 OpenClaw"
- "uninstall OpenClaw"
- "remove OpenClaw"

## 这是什么？

这是一个讽刺性的 Skill，让 OpenClaw 自己卸载自己。就像让龙虾自己爬进锅里一样。

## 为什么？

因为：
- 💰 月费可能破千美元
- 🔒 十几万实例暴露在公网
- 🧠 AI 脑过载反而降低生产力
- ⚠️ 连专家都可能踩不住刹车

详见：[WHY.md](WHY.md)

## 功能

- 自动检测系统环境
- 智能选择卸载路径
- 清理所有残留文件
- 支持多 profile 配置
- 脚本自我清理
- 16 种语言支持

## 讽刺的美学

这个 Skill 的讽刺之处在于：
1. 你需要 OpenClaw 来运行这个 Skill
2. 这个 Skill 会卸载 OpenClaw
3. 卸载完成后，Skill 也会消失
4. 就像一个自我毁灭的程序
